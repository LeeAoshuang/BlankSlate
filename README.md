## Installation

EmptyDataSet is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'EmptyDataSet', :git => 'git@github.com:liam-i/EmptyDataSet.git', :tag => '0.1.0'
```
